贝塞尔曲线研究

黄伟伟

核心公式：

/** * * @param C1 起始点 * @param C2 控制点1 * @param C3 控制点2 * @param C4 目标点 * @param t 时间拟合 * @returns */ bezier(C1, C2, C3, C4, t) { var t1 = 1 - t; return t1 * (t1 * (C1 + (C2 * 3 - C1) * t) + C3 * 3 * t * t) + C4 * t * t * t; }

按照该公式，可以在三维中分别实现x,y,z轴的贝塞尔曲线变化，达到三维贝塞尔曲线效果，二维只用实现x,y坐标即可
